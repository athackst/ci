# Contributing

This repository is maintained as a personal toolkit (similar to dotfiles) for
my own reusable GitHub Actions and workflows.

## External contributions

External pull requests are generally not accepted.

If you open a PR, it may be closed without review. This is not a reflection on
your work; it is a scope/maintenance decision for this repository.

## Issues

Issues are optional and best-effort only. If you report a bug, please include:

- What workflow or action you are using
- A minimal repro
- Relevant logs/error messages

## Forking

You are welcome to fork this repository and adapt it to your own needs.
